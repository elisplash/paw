# OpenAI

Use GPT-4o, o1, o3, and other OpenAI models.

## Setup

1. Get an API key at [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. In Pawz → **Settings → Models** → **Add Provider** → **OpenAI**
3. Paste your API key

## Configuration

| Setting | Default |
|---------|---------|
| Base URL | `https://api.openai.com/v1` |
| API key | Required |
| Default model | — |

## Models

| Model | Best for |
|-------|----------|
| `gpt-4o` | Best overall, multimodal |
| `gpt-4o-mini` | Fast, cheap, good quality |
| `o1` | Complex reasoning |
| `o3-mini` | Reasoning, cheaper |

## Prefix routing

Model names starting with `gpt*`, `o1*`, `o3*`, or `o4*` auto-route to OpenAI.
