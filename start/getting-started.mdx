# Your AI, your rules.

Pawz is a **native desktop AI agent platform** built on [Tauri v2](https://v2.tauri.app/). Create autonomous AI agents, connect them to chat platforms, and let them work — all running locally on your machine.

<Note>
Everything runs through Tauri IPC — **no open ports, no cloud dependency, no Node.js backend**. Your data stays on your machine.
</Note>

## Architecture

<Columns cols={2}>
  <Card title="Frontend — TypeScript" icon="monitor">
    - **20+ views** — chat, agents, tasks, orchestrator, settings, and more
    - **Material Symbols** icon system
    - **Vanilla TS** — no framework, fast and lightweight
    - **Tauri IPC** — all data stays local via `invoke()` calls
  </Card>
  <Card title="Backend — Rust" icon="server">
    - **Agent loop** with tool executor and streaming
    - **11 channel bridges** — Telegram, Discord, Slack, Matrix, IRC...
    - **10 AI providers** — Ollama, OpenAI, Anthropic, Google...
    - **SQLite + OS keychain** — encrypted local storage
  </Card>
</Columns>

<Note>
Everything communicates through **Tauri IPC** — no open ports, no HTTP servers, no cloud dependency. Your data never leaves your machine.
</Note>
