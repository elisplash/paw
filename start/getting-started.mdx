## What is OpenPawz?

OpenPawz is a self-hosted, native desktop app that gives you a personal AI agent platform — without giving up control of your data or relying on hosted services.

<AccordionGroup>
  <Accordion title="Who is it for?">
    Developers and power users who want AI agents they can message from anywhere — Telegram, Discord, Slack, WhatsApp — while keeping full control.
  </Accordion>
  <Accordion title="What makes it different?">
    - **Self-hosted**: runs on your hardware, your rules
    - **Multi-channel**: one app serves 11 chat platforms simultaneously
    - **Agent-native**: built for AI agents with tool use, sessions, memory, and orchestration
    - **Open source**: MIT licensed, community-driven
  </Accordion>
  <Accordion title="What do you need?">
    Rust toolchain, a supported OS (macOS / Linux / Windows), an API key or local model, and 5 minutes.
  </Accordion>
</AccordionGroup>

The desktop app is the single source of truth for agents, sessions, routing, and channel connections. Everything communicates through **Tauri IPC** — no open ports, no HTTP servers.

## Quick start

<Steps>
  <Step title="Clone and install">
    ```bash
    git clone https://github.com/elisplash/paw.git
    cd paw
    npm install
    ```
  </Step>
  <Step title="Build the Tauri app">
    ```bash
    npm run tauri dev
    ```
    The app launches with the built-in UI. No separate backend needed.
  </Step>
  <Step title="Add a provider">
    Open **Settings → Models** and add an API key for OpenAI, Anthropic, or point to a local Ollama instance.
  </Step>
  <Step title="Create your first agent">
    Go to **Agents → New Agent**, pick a model, and start chatting. Your agent is ready.
  </Step>
</Steps>

Need the full install guide? See [Installation](/start/installation).
